# Arihant Shukla - Portfolio

Welcome to the source code of my personal portfolio website.

This is a modern, responsive, and beautifully designed web portfolio built to showcase my projects, certifications, and skills as a Design Engineer and Full-Stack Developer.

## 🚀 Tech Stack

- **Frontend Framework:** React (with Vite for fast bundling)
- **Styling:** Custom CSS with Modern Glassmorphism & Micro-animations
- **Deployment:** Vercel / Netlify (Ready)

## 🛠️ Running Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Arihantshukla98/arihant-portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
